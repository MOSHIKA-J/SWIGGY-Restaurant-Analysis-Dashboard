# Swiggy Restaurant Analysis Dashboard

## Objective
To analyze Swiggy restaurant data and derive insights on city-wise demand, delivery performance, pricing, and customer engagement using Power BI.

## Tools Used
- Power BI
- Microsoft Excel

## Key KPIs
- Total Restaurants
- Cities Covered
- Customer Reviews (Count)
- Average Delivery Time (mins)
- Average Price per Order

## Dashboard Overview
This dashboard enables comparison of restaurant performance across cities and food categories to support business and operational decision-making.
